This is a simple website to chat with others.

## Preview

![](https://github.com/TsungShengWu/simple-chat/blob/main/preview.gif)

## Getting Started

1. Run the development server: `yarn dev`

2. Login with built-in accounts:

```
{
  username: 'test1',
  password: 'test1',
}
```
or
```
{
  username: 'test2',
  password: 'test2',
}
```

3. Start chatting.
